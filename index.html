☁️ Connect GitHub Sync

Your cards.json will be stored in a GitHub repository. Every device reads and writes the same file, so your SM-2 progress follows you everywhere.

  • 1
    Create a free account at github.com if you don't have one, then create a new private repository — name it anything, e.g. paeds-flashcards.
  • 2
    Go to github.com/settings/tokens/new. Create a Classic token with the repo scope ticked. Copy the token — you only see it once.
  • 3
    Fill in the fields below and click Save. Your token is stored only in this browser's localStorage — it is never sent anywhere except GitHub's own API.
